    配置别名

    git 配置别名

    例如：

    $ git status–>$ git config --global alias.st status–>$ git st

    $ git log -1–>$ git conflg --global alias.last 'log -1'–>$ git last

    --global为全局参数，不加的话仅对当前仓库有效

    仓库的配置文件在.git/config中

    用户的配置文件在$/.gitconfig中
